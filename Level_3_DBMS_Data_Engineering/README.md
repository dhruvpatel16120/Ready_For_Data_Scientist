---

## 🎯 Learning Objectives

- [ ] Write complex SQL queries with window functions and CTEs
- [ ] Design normalized database schemas (3NF, BCNF)
- [ ] Understand star/snowflake schema for data warehousing
- [ ] Process large datasets with PySpark
- [ ] Use cloud services (S3, BigQuery, Azure ML) for data science
- [ ] Containerize applications with Docker
