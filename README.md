## 🎯 Vision

> **"Contribute to the society by making quality Data Science education accessible to everyone."**

This repository is more than a personal learning journal — it's a **community resource** designed to help anyone, anywhere, become a Data Scientist through **hands-on practice**, not passive learning. Every notebook, every script, every project here is a stepping stone toward real-world mastery.

## 📖 How to Use This Repository

### 🎓 For Learners

1. **Start from Level 0** — Don't skip foundations, even if you "know" Python
2. **Follow the order** — Each level builds on the previous one
3. **Practice > Theory** — Run every notebook, modify every script, break things
4. **Build projects** — Each level has project ideas. **Do them.**
5. **Open the [Interactive Roadmap](index.html)** — Visual guide with all details

### 🔬 For Contributors

1. Read the [Contributing Guide](CONTRIBUTING.md)
2. Pick a level/topic that needs content
3. Add notebooks, scripts, or notes with clear explanations
4. Submit a Pull Request

## ⚡ Quick Start

```bash
# Clone the repository
git clone https://github.com/dhruvpatel16120/Learning_ML.git
cd Learning_ML

# Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install common dependencies
pip install numpy pandas matplotlib seaborn scikit-learn jupyter

# Launch Jupyter
jupyter notebook
```
