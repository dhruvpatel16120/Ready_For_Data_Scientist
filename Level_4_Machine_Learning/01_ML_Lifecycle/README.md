# 🔄 ML Lifecycle

- Problem framing (business → ML problem)
- Data collection (APIs, scraping, synthetic)
- EDA & preprocessing
- Model selection (baseline → iterate)
- Training & validation (k-fold CV)
- Evaluation (metrics selection)
- Hyperparameter tuning (Grid/Random/Bayesian)
- Deployment (pickle, joblib, Flask/FastAPI)
- Monitoring (data drift, model drift)
