# 🔢 NumPy — Numerical Computing

> The foundation of all numerical computing in Python

## Topics Covered
- ndarray creation (array, zeros, ones, arange, linspace, random)
- Array attributes (shape, dtype, ndim, size)
- Indexing & slicing (boolean, fancy indexing)
- Reshaping (reshape, flatten, ravel, squeeze)
- Broadcasting (CRITICAL concept)
- Mathematical operations (element-wise, dot, matmul)
- Aggregation (sum, mean, std, min, max with axis)
- Stacking & splitting (concatenate, vstack, hstack)
- Sorting & searching (sort, argsort, where, unique)
- Vectorization (why loops are slow)
- Memory views (views vs copies)
