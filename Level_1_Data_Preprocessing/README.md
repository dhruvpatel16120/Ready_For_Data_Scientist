# Level 1 Data Preprocessing

## NumPy

NumPy is a fundamental package for scientific computing in Python. It provides support for arrays, matrices, and a collection of mathematical functions to operate on these data structures. Key features include:
- **N-dimensional arrays**: A powerful object for representing complex data structures.
- **Vectorized operations**: Element-wise operations and broadcasting, which enable you to perform operations on entire arrays without looping.
- **Mathematical functions**: Built-in functions for statistical operations, linear algebra, Fourier transforms, and more.

### Learning Resources
- [NumPy Official Documentation](https://numpy.org/doc/stable/)
- [NumPy for Data Science](https://www.datacamp.com/community/tutorials/python-numpy-tutorial)

## Pandas

Pandas is an open-source data analysis and manipulation tool built on top of NumPy. It offers data structures like Series and DataFrame that are essential for data analysis. Key features include:
- **DataFrames**: A two-dimensional size-mutable, potentially heterogeneous tabular data structure with labeled axes.
- **Data manipulation**: Tools for reshaping, merging, and aggregating data.
- **Data cleaning**: Functions for handling missing data, filtering, and transforming datasets.

### Learning Resources
- [Pandas Official Documentation](https://pandas.pydata.org/docs/)
- [Pandas Guide](https://www.learnpython.com/blog/pandas/)

## Data Science Theory Fundamentals

Understanding the theoretical fundamentals of data science is crucial for effective analysis and modeling. Key areas include:
- **Statistics**: Descriptive and inferential statistics, probability theory, and statistical tests.
- **Machine Learning**: Supervised vs unsupervised learning, common algorithms, and their applications.
- **Data Exploration**: Techniques for data visualization, exploratory data analysis (EDA), and feature engineering.
