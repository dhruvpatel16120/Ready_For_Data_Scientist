## 📚 What You'll Learn

## 🎯 Learning Objectives

- [ ] Choose the right chart type for any data scenario
- [ ] Create publication-quality plots with Matplotlib
- [ ] Perform visual EDA rapidly with Seaborn
- [ ] Build interactive dashboards in Power BI
- [ ] Publish data stories on Tableau Public
- [ ] Apply Tufte's principles for clarity and impact

---

## 💡 Project Idea

> Build an end-to-end dashboard: load a real Kaggle dataset → clean with Pandas → visualize in Seaborn for EDA → build a Power BI / Tableau dashboard for business reporting.
