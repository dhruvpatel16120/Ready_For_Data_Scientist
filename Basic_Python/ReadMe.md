# 🐍 Master Python: Basic to Intermediate

Welcome to the **Basic Python** learning repository! This directory contains a curated collection of Python scripts designed to take a beginner from their first line of code to mastering essential programming concepts.

---

## 🚀 Getting Started

This repository is structured to provide hands-on examples of Python's core features. Whether you are completely new to programming or refreshing your skills, these scripts offer clear, commented demonstrations.

### 📋 Prerequisites
- **Python 3.x** installed on your system.
- An IDE (like **VS Code**, **PyCharm**, or even a terminal).

## 🛠️ How to Use This Repository

1. **Clone/Open** the directory in your VS Code.
2. **Navigate** to any file you wish to explore.
3. **Run** the script using the terminal:
   ```bash
   python <filename>.py
   ```
4. **Experiment!** Modify the variables and see how the output changes.
