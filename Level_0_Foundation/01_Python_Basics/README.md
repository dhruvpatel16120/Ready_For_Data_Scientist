## 🎯 Why Python?

In the journey to becoming a Data Scientist, Python is your most powerful tool. It's the industry standard for:

- **Data Manipulation** (Pandas/NumPy)
- **Machine Learning** (Scikit-Learn/PyTorch)
- **Generative AI** (OpenAI/LangChain)
- **General Purpose Automation**

> **"Mastering the basics is not a chore, it's the preparation for breakthroughs."**

## 📖 How to Use

1. **Clone the Repo**: Ensure you have a local copy of the project.
2. **Explore**: Open any file in the `codes/` folder.
3. **Run & Experiment**:
   ```bash
   python codes/10_strings.py
   ```
4. **Edit**: Change the values, break the logic, and see how Python reacts!
