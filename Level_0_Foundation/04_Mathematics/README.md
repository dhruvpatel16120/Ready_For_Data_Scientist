# 📐 Mathematics for Data Science

> Linear algebra, probability, statistics, calculus — the mathematical foundations of ML/AI

## Topics Covered

### Linear Algebra
- Vectors, matrices, multiplication, transpose, inverse
- Eigenvalues & eigenvectors, SVD, dot product

### Probability
- Sample space, conditional probability, Bayes theorem
- Random variables, PMF, PDF, CDF, distributions

### Statistics
- Descriptive (mean, median, mode, std, IQR)
- Inferential (hypothesis testing, p-value, confidence intervals)
- Distributions (Normal, Binomial, Poisson)

### Calculus
- Derivatives, chain rule, partial derivatives
- Gradient, gradient descent, cost function optimization
