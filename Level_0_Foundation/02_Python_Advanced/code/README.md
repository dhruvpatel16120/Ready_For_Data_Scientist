# 🐍 Mastery: Advanced Python Development

Welcome to the **Advanced Python** core. This repository is a professional-grade curriculum designed to bridge the gap between basic syntax and high-level software engineering. From the intricacies of Object-Oriented Architecture to the performance of Asynchronous I/O and Database Tuning, this project is your definitive guide to becoming a Python Power User.

---

## 🗺️ The Advanced Syllabus

The curriculum is divided into four major learning tracks, each containing hands-on modules and practical demonstrations.

### 🏛️ Track 1: Software Architecture (OOP & Modules)

Master the art of building scalable, maintainable systems.

- **[OOPS](./oops/)**: The 4 Pillars, Magic Methods, and Class Relationships.
- **[Modules](./modules/)**: Mastering the split between built-in and user-defined logic.
- **[Packages](./packages/)**: Hierarchical code organization with nested sub-packages (e.g., `ecommerce`, `swg_game`).

### 🛠️ Track 2: Systems & Operations

Handling the external world: Files, Errors, and the OS.

- **[File Handling](./file_handling/)**: Modern I/O for Text, CSV, JSON, XML, and Binary data.
- **[Exception Handling](./exception_handling/)**: Building resilient, crash-proof applications using the Exception Hierarchy.
- **[Advanced Topics](./advance_topics/)**: Exploring modern syntax like `match-case` and ternary operations.

### ⚡ Track 3: The Standard Library Power Suite

A deep dive into 30+ "batteries-included" modules for high-performance engineering.

- **[Library Masterclass](./library/)**:
  - **Concurrency**: `threading`, `multiprocessing`, `asyncio`, and `concurrent.futures`.
  - **System Control**: `os`, `sys`, `pathlib`, `shutil`, `subprocess`.
  - **Functional Tools**: `itertools`, `functools`, `collections`, `operator`.
  - **CLI & Config**: `argparse`, `configparser`, `dotenv`, `logging`.

### 🗄️ Track 4: Database Engineering (SQLite)

Full-stack data management using Python's native database engine.

- **[Database Operations](./DB_OPs/)**:
  - **Foundations**: ACID transactions, CRUD logic, and schema design.
  - **Performance**: High-speed indexing and `EXPLAIN QUERY PLAN` analysis.
  - **Extensions**: Mapping Python functions directly into SQL queries.

## 🌟 What You Get Here?

This isn't just a collection of scripts; it's a showcase of **Professional Python Standards**:

- **🛡️ Security**: Zero usage of f-strings in SQL; 100% parameterized queries.
- **🧹 Clean Code**: Strict adherence to Context Managers (`with` blocks) for resource safety.
- **🧬 Modern Patterns**: Using `pathlib` for OS-agnostic paths and `@dataclass` where appropriate.
- **🏗️ Modular Architecture**: High signal-to-noise ratio with logic separated into maintainable packages.
